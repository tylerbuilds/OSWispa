Local processing / models

Choose for the computer you have.

MorpheOS Voice checks the current build, memory and a short CPU sample before suggesting a local model. Start with that recommendation; model architecture and quantisation do not need to be first-run decisions.

02 / Download source

Models are separate downloads

MorpheOS Voice does not bundle a speech model in the application packages. Managed downloads come from the ggerganov/whisper.cpp model repository on Hugging Face and are validated before installation.

Optional model files retain their upstream licences and terms. The application's MIT licence does not relicense them. See the third-party notices.

03 / Storage

Existing models remain available

The first MorpheOS Voice transition release continues to use the established OSWispa application-data directories. This prevents a public-name change from forcing another multi-gigabyte download or breaking rollback.

On Linux, managed models normally live under ~/.local/share/oswispa/models/. macOS continues to use ~/Library/Application Support/com.oswispa.OSWispa/models/. The application resolves the correct directory on each operating system.
