Install

Get running in minutes.

Ubuntu and Debian are first class. Other platforms need manual dependencies.

Quick start

Clone and run the installer. Log out and back in to refresh group permissions.

git clone https://github.com/tylerbuilds/OSWispa.git
cd OSWispa
./install.sh
oswispa
Requires: Rust 1.70+, CMake 3.16+, libssl-dev

Auto paste tips

Wayland needs ydotoold for simulated typing. Clipboard copy is always available.

sudo ydotoold &
pgrep ydotoold
If text does not appear, start the daemon or paste manually.
